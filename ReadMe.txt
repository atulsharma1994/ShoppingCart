SHOPPING CART PROJECT
------------------------------------
To run the application :
Open the Clients package and run the main methods of each class in the following sequence:
(i)ClientsProduct.java
(ii)ClientsUser.java
(iii)ClientsOrders.java

The final Bill will be present in the Resources folder named "ServiceBill.csv" in csv format.


Supported functionalities:-
(i)The application has the functionality of adding the information of users and products in the database.
(ii)The database is updated whenever a client buys a product and confirms the order.
(iii)The quantity of product is updated whenever a client buys product.


Summary of the project:-
Shopping_Cart application is divided into four phases:
(i)Clients: This package contains the files which is used the start process of the application.
(ii)Dao: This package contains the files related to connection with data and executing sql queries.
(iii)Domain: This package contains files has getter and setter functions to set and get data from database.
(iv)Services: This package contains the files to read text files(UserData.txt,Product.txt,OrderDetail.txt).


 Programming Language Used: Java
 Database technology: MySQL
